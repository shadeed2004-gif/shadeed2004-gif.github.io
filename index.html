Flagship project · IoT + Data Fusion

WaveGuard — IoT Coastal Early Warning System

A buoy‑based monitoring system built to detect Kallakkadal — sudden, dangerous swell surges along the Kanayannur coastline, Kerala — and turn raw sensor data into early warnings for fishing communities.

  • ESP32 firmware samples MPU‑6050 IMU g‑force across 20‑sample windows, classifying sea state into CALM / WATCH / WARNING and transmitting readings every 5 seconds.
  • A custom Python fusion engine cross‑validates buoy data against the Open‑Meteo satellite wave API for more reliable alerts.
  • Bilingual (English & Malayalam) React dashboard with a JWT‑authenticated admin console.
  • GSM SMS alerts via SIM800L, with hardware tuned to handle 2A transmission spikes using a 1000µF bulk capacitor.
